=======================================================
Система контроля текущей успеваемости (СКТУ) 1.0 (2015)
=======================================================

Требования:
-----------
- PHP 5.5
- MySQL 5
- Apache 2.4 или другой HTTP-сервер


Описание:
---------
Данное веб-приложение предназначено для использования в
ВУЗах Российской Федерации в качестве автоматизированной
системы контроля текущей успеваемости студентов.

Преподаватели могут получать электронные экземпляры
ведомостей в виде Excel-таблиц для занятий и сдавать их
обратно в систему; студенты могут просматривать свои
успехи в учёбе; системный администратор следит за
актуальностью базы данных и осуществляет управление
учётными данными пользователей.

Программа распространяется бесплатно под открытым исходным
кодом под лицензией The BSD 3-Clause License:
http://opensource.org/licenses/BSD-3-Clause
